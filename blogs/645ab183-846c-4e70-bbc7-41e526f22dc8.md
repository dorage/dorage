V8은 NodeJS 와 Chrome에서 자바스크립트 엔진으로 사용되는 오픈소스 자바스크립트 엔진이다.

## V8 이 하는 일

- JS코드의 컴파일 & 실행
- Call Stack 핸들링
- Memory Heap 관리
- Garbage Collection
- Data Type 과 Object, Operators, functions 제공

## V8 이 할 수 있는 일

- Event Loop 를 제공, 그러나 브라우저가 제공하는 경우도 있음

## V8 이 하지 않는 일

- DOM과 관련된 일, 이는 브라우저가 제공한다.
